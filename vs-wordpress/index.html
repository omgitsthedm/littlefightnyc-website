The plugin rot problem

Free plugins go unmaintained. Your site pays.

A typical WordPress site for a NYC small business loads 15–30 plugins — SEO, security, contact form, page builder, e-commerce, backup, image optimization, social sharing. Each plugin is written by an independent developer. The "free" plugin you installed today might be:

  • Abandoned by its author in 18 months — no security updates, opens vulnerabilities
  • Acquired by a company that turns it paid (Yoast SEO, WPBakery, etc.)
  • Replaced by a "Pro" version that requires a $50–$200/year license
  • Incompatible with the next WordPress core update — takes the site down
  • Slowing the site down with every page load (most are not optimized)

The hidden cost: a real ongoing budget for plugin licenses ($300–$1,000/year) and a developer or agency on retainer to handle conflicts. Custom-coded sites don’t have this problem.

When each wins

Honest read on which fits.

Pick WordPress if:

  • You’re running a content-heavy site (blog, magazine, news)
  • You have a developer or agency on retainer to maintain it
  • You need a deep CMS with custom post types, taxonomies, etc.
  • You’re technically comfortable managing plugin updates
  • You’re fine with $300–$1,000/year in plugin licenses

Pick Little Fight if:

  • You need service pages + booking + contact + maybe a store
  • You don’t want to manage plugin updates or developer retainers
  • You want it secure by default — no CMS attack surface
  • You want it fast — Lighthouse 95–100, not 40–70
  • You want a real human to call when something breaks

Owner FAQ

Questions owners ask first.

Is WordPress good for a NYC small business website?

Self-hosted WordPress.org with a custom theme is fine if you have a developer. WordPress.com hosted plans ($25–$70/month) are slow, plugin-bloated, and locked-in. Most NYC small businesses end up with a WordPress site loaded with 15–30 plugins, each a security and performance risk. Custom-built outperforms typical WordPress sites on every metric.

What’s plugin rot?

WordPress plugins are written by independent developers. They go unmaintained, get abandoned, or get replaced by paid versions. The "free" plugin you used today might break the site in 18 months when its author stops updating it — and may take your site down with it. Custom code doesn’t have this problem.

What does WordPress.com really cost over 5 years?

WordPress.com Business is $25/month — $1,500 over 5 years. Add typical paid plugins (security, SEO, backup, page builder, e-commerce) at $200–$400/year and you’re at $2,500–$3,500. Plus your time managing updates and plugin conflicts. See the lifetime cost calculator for your specific tier.

Does Google rank WordPress sites well?

Self-hosted WordPress with a fast, custom-coded theme can rank well. Most WordPress.com hosted sites (especially with page builders like Elementor or Divi) load slowly and underperform on Core Web Vitals. The platform isn’t penalized — slow page-builder sites are.

Is WordPress ever the right answer?

Yes — for a content-heavy site (blog, magazine, news) where you genuinely need a CMS and have a developer to maintain it. For most NYC small businesses (service pages, booking, contact), WordPress is overkill and the plugin maintenance cost adds up fast. We’ll tell you that honestly.
