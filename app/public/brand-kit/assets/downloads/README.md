# Little Fight NYC Brand Kit

Version 2.0.0, updated July 24, 2026.

Open `index.html` in a browser for the visual guide. The archive contains its
fonts, imagery, styles, script, identity assets, design tokens, token schema,
and written operating guide, so the page works without a network connection.

## Start here

1. Open `index.html`.
2. Read `BRAND-KIT.md` for the concise rules.
3. Use `color/palette.css` or `color/tokens.json` when building.
4. Use the fixed-color logo file that matches the intended ground.
5. Use `logo/mark-currentcolor.svg` only for inline CSS-controlled SVG.
