# Canva Brand Kit: Little Fight NYC (Ocean Ember)

## Brand Direction
- Keep original bright orange energy for hero/UI impact.
- Use blue for readability, links, small text, and accessibility.

## Core Colors (Add To Canva Brand Kit)
- Brand Orange (Primary Visual): `#FE5800`
- Ocean Text Blue (Accessible): `#0C6A84`
- Ocean Action Blue: `#0891B2`
- Ocean Hover Blue: `#0E7490`
- Light Warm Tint: `#FFF9F5`
- Light Cool Tint: `#F0F9FF`
- Charcoal Text: `#333333`
- Secondary Text: `#595959`
- Muted Text: `#767676`
- Dark Surface: `#222222`
- Footer Text: `#B8BFC8`
- White: `#FFFFFF`

## Usage Split (Important)
- Orange `#FE5800`: buttons, logo box, hero shapes, large highlights, CTA blocks.
- Blue `#0C6A84` and `#0891B2`: links, labels, section eyebrows, icon accents, form focus, small text accents.

## Recommended Gradients
- Hero wash: `#FFF9F5 -> #FFFFFF -> #F0F9FF`
- Signature CTA gradient: `#FE5800 -> #0891B2`

## Typography (Canva)
- Heading font: `Lexend` (700/800)
- Body font: `Inter` (400/500/600)
- Optional accent script: `Caveat` (600)

## Icon Style
- Rounded line icons, simple and geometric.
- Default icon stroke: `#0C6A84`
- Highlight icon stroke/fill accents: `#0891B2`
- On orange/blue solid blocks: `#FFFFFF`

## Website Token Alignment
- `--accent-ui: #FE5800`
- `--accent-text: #0C6A84`
- `--accent-blue-bright: #0891B2`
- `--accent-blue-dark: #0E7490`

## Canva Setup Checklist
1. Add all colors in Core Colors.
2. Set `Lexend` for headings and `Inter` for body.
3. Save two button styles:
   - Primary: orange fill, white text.
   - Secondary: blue outline/text, light blue hover fill.
4. Save one icon style (blue stroke, rounded line set).
